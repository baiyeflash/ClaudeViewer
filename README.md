# Claude 数据查看器 · 使用说明书

> 版本 v2.0 · 本地运行 · 零上传 · 单文件 HTML

---

## 一、软件简介

**Claude 数据查看器**是一款完全在浏览器本地运行的数据阅读工具，专为 Claude.ai 导出的个人数据设计。无需安装，无需联网，双击 HTML 文件即可使用。

**核心特性**
- 🔒 数据完全本地处理，不上传任何内容到服务器
- ⚡ 虚拟滚动技术，万条消息流畅浏览
- 📦 支持直接导入 ZIP 导出包，一键解析全部数据
- 🧠 支持查看对话、项目、记忆、账户四类数据
- 📝 支持导入自定义 Markdown 记忆文件

---

## 二、如何获取 Claude 导出数据

1. 打开 [claude.ai](https://claude.ai)，登录账户
2. 点击右上角头像 → **Settings（设置）**
3. 找到 **Privacy（隐私）** → **Export data（导出数据）**
4. 选择时间范围，点击 **Export** 按钮
5. 等待邮件通知（通常几分钟内），下载 `.zip` 压缩包

---

## 三、快速开始

### 3.1 打开软件
双击 `claude_viewer.html` 文件，在任意现代浏览器中打开（推荐 Chrome / Edge / Firefox）。

### 3.2 导入数据
支持三种方式：

| 方式 | 操作 |
|------|------|
| 拖放文件 | 直接将文件拖入页面中央的拖放区域 |
| 点击选择 | 点击「选择文件」按钮，从文件管理器中选取 |
| 混合导入 | 同时选择多个文件（JSON + ZIP + MD）一次性导入 |

支持的文件类型：
- `.zip` — Claude 导出的完整压缩包（推荐，自动解析全部内容）
- `conversations.json` — 对话记录
- `users.json` — 账户信息
- `memories.json` — 记忆数据
- `projects/*.json` — 项目文件
- `.md` / `.markdown` — 自定义 Markdown 记忆文件

## 四、功能详解

### 4.1 💬 对话（Conversations）

**对话列表**
- 左侧显示所有有内容的对话，按最后活跃时间倒序排列
- 每条显示：对话标题 · 消息数量 · 最后活跃时间
- 顶部搜索框支持按标题筛选
- 空对话（无消息内容）自动过滤，不展示

**对话内容**
- 点击任意对话卡片进入详情页
- Human 消息气泡靠右（橙米色背景），Claude 消息气泡靠左（白色边框）
- 每条消息显示完整时间戳（年-月-日 时:分）
- Claude 的回复支持完整 Markdown 渲染（标题、代码块、表格、引用等）

**思考过程（Thinking）**
- 支持显示 Claude Extended Thinking 模式的思考内容
- 默认折叠，点击展开查看完整推理过程
- 折叠状态下显示思考摘要预览
- 思考内容以浅米色背景 + 斜体样式与正式回复视觉区分

**关键词搜索**
1. 在对话详情页顶部输入框输入关键词
2. 点击「搜索」按钮或按 Enter 执行搜索
3. 所有匹配位置黄色高亮，当前定位橙色高亮
4. 使用 ↑ / ↓ 按钮在结果间跳转
5. 右侧显示「当前 / 总数」计数

**虚拟滚动**
软件采用虚拟滚动技术，无论对话有多少条消息，始终只渲染可视区域内的内容，保证长对话流畅浏览。

### 4.2 📁 项目（Projects）

显示 Claude Projects 的全部元数据，左侧列出所有项目，点击查看详情：

| 内容 | 说明 |
|------|------|
| 基本信息 | 项目名称、创建/更新时间、可见性 |
| 系统指令 | 该项目的 System Prompt 全文 |
| 知识库文档 | 项目内上传的文件内容 |
| 项目记忆 | 该项目独立维护的记忆内容 |

### 4.3 🧠 记忆（Memories）

记忆数据分两类来源：

**导入 memories.json（项目记忆）**
- 解析 memories.json 后自动显示
- 按项目名称分组，左侧列表点击切换

**导入 Markdown 全局记忆**
- 点击记忆 Tab 左侧顶部的「＋ 导入全局记忆 .md」按钮
- 或在初始导入时直接拖入 .md 文件
- 支持同时导入多个 .md 文件
- 右侧以完整 Markdown 渲染显示
- 右上角「↓ 导出」按钮可重新下载该文件

> 💡 **提示**：可将 Claude 聊天界面「导出记忆」生成的 Markdown 文件直接导入此处查看。

### 4.4 👤 账户（Account）

显示 users.json 中的账户基本信息及统计数据：
- 用户名、邮箱、手机号、账户 UUID
- 对话总数、项目总数、总消息数、记忆状态

## 五、导出功能

在对话详情页右上角有两个导出按钮：

### ↓ Markdown
将当前对话导出为 `.md` 文件，包含：
- 完整对话内容，保留 Markdown 格式
- 每条消息的发送者标签和时间戳
- 思考过程以 `<details>` 折叠块形式保留

### ↓ PDF
在新窗口中渲染排版好的对话页面，并自动触发打印对话框：
1. 在打印对话框中选择「目标打印机」→「另存为 PDF」
2. 点击保存即可获得 PDF 文件

## 六、快捷键

| 操作 | 快捷键 |
|------|--------|
| 执行搜索 | `Enter` |
| 下一个搜索结果 | `Enter`（已有结果时）或点击 ↓ |
| 上一个搜索结果 | 点击 ↑ |

---

## 七、注意事项

1. **浏览器兼容性**：推荐使用 Chrome 90+、Edge 90+、Firefox 88+，不支持 IE
2. **PDF 导出**：需要允许浏览器弹出新窗口（部分浏览器会默认拦截弹出窗口）
3. **大文件**：首次解析大型 ZIP 包（数千条对话）可能需要数秒，请耐心等待进度提示
4. **隐私安全**：所有数据仅在本地浏览器内存中处理，刷新页面后数据即清除，不会存储在任何地方

## 八、数据说明

Claude 导出包中各文件的含义：

| 文件 | 内容 |
|------|------|
| `conversations.json` | 全部对话记录（消息内容、时间戳、Thinking 数据） |
| `users.json` | 账户基本信息（姓名、邮箱、手机号） |
| `memories.json` | Claude 的记忆数据（项目记忆） |
| `projects/*.json` | 各项目元数据（名称、System Prompt、文档、记忆） |
