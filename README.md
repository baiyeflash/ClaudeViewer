# 🗂️ Claude 数据查看器 v3.0

> 本地运行 · 零安装 · 数据不离开你的设备

一个单文件 HTML 工具，用于查看和分析从 Claude.ai 导出的个人数据。双击即用，无需服务器、无需网络、无需账号。

---

|---|---|

## 🚀 快速开始

### 第一步：获取 Claude 导出数据

1. 打开 [claude.ai](https://claude.ai) 并登录
2. 点击右上角头像 → **Settings** → **Privacy** → **Export data**
3. 点击 **Export**，等待邮件通知（通常几分钟内）
4. 下载邮件中的 `.zip` 压缩包

### 第二步：打开查看器

双击 `claude_viewer_v3.html`，在浏览器中打开即可。

> **推荐浏览器**：Chrome / Edge  
> Safari 可正常查看对话，但 PDF 导出功能受限

### 第三步：导入数据

将 `.zip` 文件**直接拖入**页面中央区域，自动解析全部内容。

或选择单独导入：
- 拖入 `conversations.json` 查看对话
- 拖入 `projects/*.json` 查看项目
- 拖入 `.md` 文件查看全局记忆

## 📖 详细说明

### 对话列表

- 按最后活跃时间倒序排列
- 卡片显示：标题 · 消息数 · 思考块数 · 附件数 · 分支标识 · 时间
- 点击 ☆ 收藏，点击「+ 标签」添加分类标签
- 顶部筛选：⭐ 仅收藏 · 🏷 按标签 · 标题关键词

### 对话详情

进入对话后：
- 顶部显示对话标题，左侧「← 返回」回到列表
- 若有分支，显示「🌿 X条分支」按钮，点击展开切换面板
- 搜索栏输入关键词，回车执行，↑↓ 跳转结果

### 思考过程

Claude Extended Thinking 的推理内容：
- 默认折叠，折叠状态显示摘要预览
- 点击展开查看完整思考链
- 浅米色背景斜体，与正式回复视觉区分
- 全局搜索和对话内搜索均可搜索思考内容

### 记忆 Tab

**项目记忆**（来自 `memories.json`）：各项目独立维护的记忆，按项目名分类

**全局记忆**（`.md` 文件）：
- 在 Claude 界面导出全局记忆为 `.md` 后，拖入查看器
- 支持同时导入多个 `.md` 文件
- 完整 Markdown 渲染，右上角可重新导出

### 收藏与标签

数据存储在浏览器 `localStorage`，刷新页面不丢失：
- **收藏**：点击卡片右上角 ☆，再次点击取消
- **标签**：点击「+ 标签」输入，Enter 确认，Escape 取消
- **删除标签**：点击已有标签即删除
- **筛选**：顶部 filter chips 点击激活

## 🔒 隐私说明

- **完全本地运行**：所有数据仅在浏览器内存中处理，不经过任何服务器
- **网络请求**：仅加载两个 CDN 库（marked.js、JSZip），不传输任何个人数据
- **数据生命周期**：刷新页面即清除，不留痕迹
- **localStorage 存储内容**：收藏列表、标签数据、深色模式偏好（不含对话内容）

## 📦 技术栈

- 纯原生 HTML / CSS / JavaScript，无框架，无构建工具
- [marked.js 9.1.6](https://marked.js.org/) — Markdown 渲染
- [JSZip 3.10.1](https://stuk.github.io/jszip/) — ZIP 文件解析与生成
- 虚拟滚动：自实现（绝对定位 + requestAnimationFrame）
- 统计图表：内联 SVG，无第三方图表库
- 深色模式：CSS 变量 + `data-theme` 属性切换

## 📁 导出包文件说明

| 文件路径 | 内容 |
|---|---|
| `conversations.json` | 全部对话（消息内容、时间戳、Thinking 块、附件） |
| `users.json` | 账户基本信息（姓名、邮箱、手机号） |
| `memories.json` | 项目记忆数据 |
| `projects/{uuid}.json` | 项目元数据（名称、System Prompt、知识库文档） |

> **全局记忆说明**：Claude 跨对话记住的用户信息目前不在标准导出包内，需在 Claude 界面单独导出为 `.md` 文件后导入查看器的记忆 Tab 查看。
